# Life Mentor

## AI-Based Lifestyle & Wellbeing Mentor Web Application

Life Mentor is an AI-powered web application designed to help individuals improve their daily lifestyle and overall wellbeing. The system acts as a digital mentor by analyzing user habits related to sleep, meals, exercise, study, and mental wellbeing, and providing personalized, supportive, and non-medical guidance using artificial intelligence.

---

## Project Description

In today’s fast-paced lifestyle, many people struggle to maintain healthy routines due to stress, irregular schedules, and lack of proper guidance. Life Mentor addresses this problem by offering an intelligent web-based solution that encourages healthier habits through daily check-ins and AI-generated feedback.

The application focuses on general wellbeing support and self-improvement. It does not provide medical or psychological diagnoses.

---

## Features

- User registration and secure login
- Initial lifestyle assessment questionnaire
- Daily lifestyle check-ins (meals, exercise, study, mood)
- AI-based personalized feedback and motivation
- Daily and weekly progress tracking dashboard
- Responsive and modern user interface
- Ethical and non-medical AI guidance with disclaimers

---

## Technology Stack

### Frontend
- React.js with Vite
- Tailwind CSS
- Framer Motion

### Backend
- Java Spring Boot
- Spring Security
- RESTful APIs

### Database
- MySQL

### Artificial Intelligence
- AI Language Model API (e.g., OpenAI / Gemini)

---

## System Architecture

User → React + Vite Frontend → Spring Boot REST API → MySQL Database → AI Language Model API

---

## Ethical Considerations

- The system provides general lifestyle guidance only
- No medical or psychological diagnosis is performed
- User data is stored securely and confidentially
- Clear disclaimers are displayed to users
- AI responses are designed to be safe and supportive

---

## Disclaimer

Life Mentor is not a medical or mental health diagnostic tool.  
The guidance provided by this application should not replace professional medical or psychological advice.
